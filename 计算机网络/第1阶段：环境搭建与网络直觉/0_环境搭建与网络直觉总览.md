# 0. 环境搭建与网络直觉总览

本阶段目标：搭建可重复的网络学习环境，建立实验规范，并完成从本机信息、容器端口映射、抓包到 Go HTTP 服务的第一次端到端验证。

后续所有协议实验都依赖稳定环境和证据记录。先把工具、命令、代码与抓包串起来，能避免只记概念却不会验证。

---

## 一、本阶段学习顺序

1. `1. 学习方法与实验规范`
2. `2. 安装 Go、Docker 与基础网络工具`
3. `3. 认识本机网络信息：IP、网关、DNS、端口`
4. `4. 使用 Docker 理解容器网络和端口映射`
5. `5. 使用 Wireshark 与 tcpdump 做第一次抓包`
6. `6. 启动一个 Go HTTP 服务并完成端到端访问`

请按顺序完成正文、命令或代码实验，并使用每节末尾的达标标准验收。

---

## 二、本阶段达标标准

- 能用自己的话复述本阶段核心原理，而不是只背术语。
- 能运行章节中的关键命令或 Go 示例，并记录实际输出。
- 能把抓包、日志、连接状态或错误信息作为判断依据。
- 能说明这些知识在 Go 后端开发和线上排障中的具体用途。
