# 0. 项目实战总览

本阶段目标：通过 4 个项目把计算机网络知识落到 Go 后端工程中：TCP 聊天室、HTTP 反向代理、迷你 API 网关、gRPC 用户服务。

---

## 一、项目顺序

```text
1_TCP聊天室项目总览与协议设计
2_TCP聊天室_连接管理_广播_心跳
3_HTTP反向代理项目总览与单上游转发
4_HTTP反向代理_负载均衡_超时_日志
5_迷你API网关_路由_限流_健康检查
6_gRPC用户服务_proto_server_client
7_项目验收_排障_复盘清单
```

---

## 二、本阶段达标标准

完成后你应该能够：

- 管理 TCP 长连接。
- 处理 TCP 消息边界。
- 写 HTTP 反向代理。
- 设置超时和连接池。
- 设计迷你网关。
- 写 gRPC Server 和 Client。
- 为项目写 README、运行命令、测试命令和排障记录。
