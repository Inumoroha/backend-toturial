# 4. 项目四：高性能日志缓冲器

本项目目标：实现一个多 goroutine 写入、后台批量刷出的日志缓冲器，练习锁、WaitGroup、Pool 和关闭流程。

会使用：

- `sync.Mutex`
- `sync.WaitGroup`
- `sync.Pool`
- channel
- `context.Context`

---

## 一、需求分析

日志缓冲器需要支持：

```text
多 goroutine 写日志；
内存中暂存日志；
达到批量大小或定时触发时刷出；
使用 sync.Pool 复用 buffer；
Close 时刷出剩余日志；
后台 goroutine 能优雅退出。
```

---

## 二、结构设计

```go
type Logger struct {
    mu      sync.Mutex
    entries []string

    flushSize int
    interval  time.Duration

    stop chan struct{}
    wg   sync.WaitGroup

    pool sync.Pool
}
```

共享状态：

- `entries`

生命周期控制：

- `stop`
- `wg`

对象复用：

- `pool`

---

## 三、Write

```go
func (l *Logger) Write(msg string) {
    l.mu.Lock()
    l.entries = append(l.entries, msg)
    shouldFlush := len(l.entries) >= l.flushSize
    l.mu.Unlock()

    if shouldFlush {
        l.Flush()
    }
}
```

注意：是否要在锁外调用 `Flush` 需要设计清楚。

更简单的版本可以让 `Flush` 自己重新加锁并交换 slice。

---

## 四、Flush

```go
func (l *Logger) Flush() {
    l.mu.Lock()
    if len(l.entries) == 0 {
        l.mu.Unlock()
        return
    }

    entries := l.entries
    l.entries = nil
    l.mu.Unlock()

    buf := l.pool.Get().(*bytes.Buffer)
    buf.Reset()
    defer l.pool.Put(buf)

    for _, entry := range entries {
        buf.WriteString(entry)
        buf.WriteByte('\n')
    }

    // write buf to file/stdout/remote sink
}
```

关键点：

```text
锁内只交换 entries；
锁外执行格式化和 IO；
buffer 使用后放回 Pool。
```

---

## 五、后台定时刷出

```go
func (l *Logger) start() {
    l.wg.Add(1)
    go func() {
        defer l.wg.Done()

        ticker := time.NewTicker(l.interval)
        defer ticker.Stop()

        for {
            select {
            case <-ticker.C:
                l.Flush()
            case <-l.stop:
                l.Flush()
                return
            }
        }
    }()
}
```

---

## 六、Close

```go
func (l *Logger) Close() {
    close(l.stop)
    l.wg.Wait()
}
```

真实项目中要让 `Close` 可重复调用，可以使用：

- `sync.Once`
- `atomic.Bool + CompareAndSwap`
- `Mutex` 保护 closed 状态

---

## 七、测试清单

必须测试：

- 并发 `Write` 无 race。
- 达到批量大小会刷出。
- 定时会刷出。
- `Close` 会刷出剩余日志。
- `Close` 后后台 goroutine 退出。
- `Pool` 使用前会 `Reset`。
