# 6. 详情、列表与分页

本节目标：实现短链接详情查询和列表查询。

---

## 一、详情接口

```text
GET /links/{code}
```

返回：

```json
{
  "code": "aB39xQ",
  "original_url": "https://example.com",
  "short_url": "http://localhost:8080/aB39xQ",
  "visit_count": 12
}
```

不存在返回：

```text
404 Not Found
```

---

## 二、列表接口

```text
GET /links?page=1&page_size=20
```

响应：

```json
{
  "items": [],
  "page": 1,
  "page_size": 20
}
```

内存版本可以先简单返回全部再切片。数据库版本应该使用 SQL 分页。

---

## 三、分页参数校验

```go
page := parsePositiveInt(r.URL.Query().Get("page"), 1)
pageSize := parsePositiveInt(r.URL.Query().Get("page_size"), 20)
if pageSize > 100 {
	pageSize = 100
}
```

不要允许客户端传特别大的 page_size。
