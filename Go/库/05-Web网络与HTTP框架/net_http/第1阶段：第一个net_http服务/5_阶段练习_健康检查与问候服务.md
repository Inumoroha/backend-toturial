# 5. 阶段练习：健康检查与问候服务

本节目标：把第 1 阶段知识串起来，完成一个小服务。它不复杂，但要你完整经历“写路由、读请求、写响应、用 curl 验证”的闭环。

---

## 一、需求

实现一个 HTTP 服务，包含下面接口：

```text
GET /health
GET /hello
GET /hello/{name}
GET /debug
GET /search?q=keyword
```

接口行为：

- `/health` 返回 `ok`。
- `/hello` 返回 `hello, net/http`。
- `/hello/{name}` 返回 `hello, {name}`。
- `/debug` 返回 method、path、remote address。
- `/search` 必须传 `q`，否则返回 `400`。

---

## 二、参考实现

```go
package main

import (
	"fmt"
	"log"
	"net/http"
)

func main() {
	mux := http.NewServeMux()

	mux.HandleFunc("GET /health", healthHandler)
	mux.HandleFunc("GET /hello", helloHandler)
	mux.HandleFunc("GET /hello/{name}", helloNameHandler)
	mux.HandleFunc("GET /debug", debugHandler)
	mux.HandleFunc("GET /search", searchHandler)

	log.Println("server listening on :8080")
	log.Fatal(http.ListenAndServe(":8080", mux))
}

func healthHandler(w http.ResponseWriter, r *http.Request) {
	fmt.Fprintln(w, "ok")
}

func helloHandler(w http.ResponseWriter, r *http.Request) {
	fmt.Fprintln(w, "hello, net/http")
}

func helloNameHandler(w http.ResponseWriter, r *http.Request) {
	name := r.PathValue("name")
	fmt.Fprintf(w, "hello, %s\n", name)
}

func debugHandler(w http.ResponseWriter, r *http.Request) {
	fmt.Fprintf(w, "method=%s\n", r.Method)
	fmt.Fprintf(w, "path=%s\n", r.URL.Path)
	fmt.Fprintf(w, "remote_addr=%s\n", r.RemoteAddr)
}

func searchHandler(w http.ResponseWriter, r *http.Request) {
	q := r.URL.Query().Get("q")
	if q == "" {
		http.Error(w, "missing q", http.StatusBadRequest)
		return
	}

	fmt.Fprintf(w, "search: %s\n", q)
}
```

## 三、验证命令

启动：

```bash
go run .
```

验证：

```bash
curl -i http://localhost:8080/health
curl -i http://localhost:8080/hello
curl -i http://localhost:8080/hello/alice
curl -i http://localhost:8080/debug
curl -i "http://localhost:8080/search?q=golang"
curl -i http://localhost:8080/search
```

最后一个请求应该返回 `400 Bad Request`。

## 四、你需要自己改进的地方

在参考实现基础上，继续完成：

1. 给 `/debug` 增加 `user_agent` 输出。

4. 访问不存在路径，观察返回什么。
