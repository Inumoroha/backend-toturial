# 5. 阶段练习：简单文件服务

本节目标：实现一个最小文件服务，包含上传、下载和静态文件访问。

---

## 一、需求

接口：

```text
POST /upload
GET  /files/{name}
GET  /static/*
```

要求：

- 上传大小限制 5MB。
- 只允许 `.jpg`、`.jpeg`、`.png`、`.gif`。
- 保存时生成新文件名。
- 下载时防止路径穿越。
- 静态文件目录使用 `public`。

---

## 二、目录结构

```text
file-server/
  go.mod
  main.go
  uploads/
  public/
    index.html
```

---

## 三、验证命令

上传：

```bash
curl -i -X POST http://localhost:8080/upload `
  -F "file=@./avatar.png"
```

下载：

```bash
curl -i http://localhost:8080/files/上传后返回的文件名
```

静态文件：

```bash
curl -i http://localhost:8080/static/index.html
```

错误场景：

```bash
curl -i -X POST http://localhost:8080/upload
curl -i -X POST http://localhost:8080/upload -F "file=@./large-file.zip"
curl -i http://localhost:8080/files/../../secret.txt
```
