## 推荐节奏

如果你每天学习 1 到 2 小时，建议用 10 到 12 周完成。

如果你每天学习 3 小时以上，可以用 8 周完成，但不建议压缩实战项目时间。

## 最终目标

完成后你应该能够独立开发一个中小型 Go 后端服务，包括：

- 使用 Gin 设计 RESTful API。
- 编写清晰的 handler、service、repository 分层代码。
- 使用 GORM 操作数据库。
- 实现注册、登录、JWT 认证和权限控制。
- 编写中间件、参数校验、统一响应和错误处理。
- 添加配置、日志、测试和接口文档。
- 使用 Redis 处理缓存、限流等场景。
- 使用 Docker Compose 部署服务。

学习过程中建议保留每个阶段的代码版本。你可以为每个阶段创建一个 Git commit，这样后面回看会非常清楚。
