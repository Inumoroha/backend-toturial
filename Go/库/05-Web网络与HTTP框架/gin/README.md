# Gin 完整教程

这套教程把原有两套 Gin 课程合并为一条单一学习路径：从 Go 与 HTTP 开始，经过 Gin 路由、接口契约、认证、工程分层、GORM、测试、Redis 和 Docker 部署，最终完成一个可验收的任务管理系统后端。

## 使用方式

按文件编号顺序学习。每个阶段都从 `0_` 总览进入，其中已合并该阶段的学习目标、工程边界与实践补充；随后按编号逐节完成代码、验证命令与验收项。建议每一阶段完成后提交一个 Git commit，保留可运行版本。

## 最终产出

完成全部 92 节详细课后，你将拥有一个可部署的任务管理系统后端：包含版本化 REST API、统一响应与校验、JWT 认证、角色与资源授权、GORM/MySQL、Redis 缓存与限流、结构化日志、测试、API 文档、Docker Compose 和 Nginx 部署配置。
