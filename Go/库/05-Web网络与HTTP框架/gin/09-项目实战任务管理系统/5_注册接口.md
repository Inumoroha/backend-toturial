# 5. 注册接口

本节目标：实现 `POST /api/v1/auth/register`，完成参数校验、邮箱唯一检查、密码加密和用户创建。

---

## 一、注册流程

注册接口的完整流程：

```text
读取 JSON 请求
→ 校验 username/email/password
→ 查询邮箱是否已存在
→ bcrypt 加密密码
→ 写入 users 表
→ 返回用户基本信息
```

注意：永远不要保存明文密码。

## 二、安装 bcrypt

```bash
go get golang.org/x/crypto/bcrypt
```

创建 `internal/service/password.go`：

```go
package service

import "golang.org/x/crypto/bcrypt"

func HashPassword(password string) (string, error) {
    bytes, err := bcrypt.GenerateFromPassword([]byte(password), bcrypt.DefaultCost)
    if err != nil {
        return "", err
    }
    return string(bytes), nil
}

func CheckPassword(password, hash string) bool {
    return bcrypt.CompareHashAndPassword([]byte(hash), []byte(password)) == nil
}
```

## 三、UserRepository

`internal/repository/user_repository.go`：

```go
package repository

import (
    "errors"

    "task-api/internal/model"

    "gorm.io/gorm"
)

var ErrUserNotFound = errors.New("user not found")

type UserRepository struct {
    db *gorm.DB
}

func NewUserRepository(db *gorm.DB) *UserRepository {
    return &UserRepository{db: db}
}

func (r *UserRepository) Create(user *model.User) error {
    return r.db.Create(user).Error
}

func (r *UserRepository) GetByEmail(email string) (*model.User, error) {
    var user model.User
    err := r.db.Where("email = ?", email).First(&user).Error
    if errors.Is(err, gorm.ErrRecordNotFound) {
        return nil, ErrUserNotFound
    }
    if err != nil {
        return nil, err
    }
    return &user, nil
}
```

## 四、AuthService 注册逻辑

`internal/service/auth_service.go`：

```go
package service

import (
    "errors"

    "task-api/internal/model"
    "task-api/internal/repository"
)

var ErrEmailAlreadyExists = errors.New("email already exists")

type AuthService struct {
    userRepo *repository.UserRepository
}

func NewAuthService(userRepo *repository.UserRepository) *AuthService {
    return &AuthService{userRepo: userRepo}
}

type RegisterInput struct {
    Username string
    Email    string
    Password string
}

func (s *AuthService) Register(input RegisterInput) (*model.User, error) {
    _, err := s.userRepo.GetByEmail(input.Email)
    if err == nil {
        return nil, ErrEmailAlreadyExists
    }
    if err != nil && !errors.Is(err, repository.ErrUserNotFound) {
        return nil, err
    }

    hash, err := HashPassword(input.Password)
    if err != nil {
        return nil, err
    }

    user := &model.User{
        Username:     input.Username,
        Email:        input.Email,
        PasswordHash: hash,
        Role:         "user",
    }

    if err := s.userRepo.Create(user); err != nil {
        return nil, err
    }

    return user, nil
}
```

## 五、AuthHandler

`internal/handler/auth_handler.go`：

```go
package handler

import (
    "errors"
    "net/http"

    "task-api/internal/response"
    "task-api/internal/service"

    "github.com/gin-gonic/gin"
)

type AuthHandler struct {
    service *service.AuthService
}

func NewAuthHandler(service *service.AuthService) *AuthHandler {
    return &AuthHandler{service: service}
}

type RegisterRequest struct {
    Username string `json:"username" binding:"required,min=2,max=32"`
    Email    string `json:"email" binding:"required,email"`
    Password string `json:"password" binding:"required,min=8,max=64"`
}

func (h *AuthHandler) Register(c *gin.Context) {
    var req RegisterRequest
    if err := c.ShouldBindJSON(&req); err != nil {
        response.Fail(c, http.StatusBadRequest, response.CodeInvalidRequest, "invalid request")
        return
    }

    user, err := h.service.Register(service.RegisterInput{
        Username: req.Username,
        Email:    req.Email,
        Password: req.Password,
    })
    if errors.Is(err, service.ErrEmailAlreadyExists) {
        response.Fail(c, http.StatusConflict, response.CodeConflict, "email already exists")
        return
    }
    if err != nil {
        response.Fail(c, http.StatusInternalServerError, response.CodeInternalError, "internal server error")
        return
    }

    response.Created(c, user)
}
```

## 六、路由注册

`internal/router/router.go`：

```go
package router

import (
    "task-api/internal/handler"

    "github.com/gin-gonic/gin"
)

func New(authHandler *handler.AuthHandler) *gin.Engine {
    r := gin.Default()

    api := r.Group("/api/v1")
    {
        api.POST("/auth/register", authHandler.Register)
    }

    return r
}
```

## 八、常见问题

### 1. 密码被返回了

检查 `User.PasswordHash` 是否写了：

```go
json:"-"
```

### 2. 邮箱重复返回 500

检查 service 是否把邮箱已存在转换为 `ErrEmailAlreadyExists`。

### 3. 参数错误没有触发

检查请求结构体是否写了 `binding` tag。
