# 10. 任务更新、状态修改与删除

本节目标：完成任务模块剩余接口：整体更新、状态修改和删除。

---

## 一、接口清单

```text
PUT    /api/v1/tasks/:id
PATCH  /api/v1/tasks/:id/status
DELETE /api/v1/tasks/:id
```

这三个接口都必须认证，并且只能操作当前用户自己的任务。

---

## 二、Repository 更新和删除

```go
func (r *TaskRepository) Update(task *model.Task) error {
    return r.db.Save(task).Error
}

func (r *TaskRepository) Delete(id, userID uint) error {
    result := r.db.Where("id = ? AND user_id = ?", id, userID).Delete(&model.Task{})
    if result.Error != nil {
        return result.Error
    }
    if result.RowsAffected == 0 {
        return ErrTaskNotFound
    }
    return nil
}
```

删除时带 `user_id`，避免删除别人的任务。

## 三、Service 更新

```go
type UpdateTaskInput struct {
    UserID      uint
    TaskID      uint
    Title       string
    Description string
    Status      string
}

func (s *TaskService) Update(input UpdateTaskInput) (*model.Task, error) {
    if !isValidTaskStatus(input.Status) {
        return nil, ErrInvalidTaskStatus
    }

    task, err := s.repo.GetByIDAndUserID(input.TaskID, input.UserID)
    if err != nil {
        return nil, err
    }

    task.Title = input.Title
    task.Description = input.Description
    task.Status = input.Status

    if err := s.repo.Update(task); err != nil {
        return nil, err
    }

    return task, nil
}

func (s *TaskService) UpdateStatus(userID, taskID uint, status string) (*model.Task, error) {
    if !isValidTaskStatus(status) {
        return nil, ErrInvalidTaskStatus
    }

    task, err := s.repo.GetByIDAndUserID(taskID, userID)
    if err != nil {
        return nil, err
    }

    task.Status = status
    if err := s.repo.Update(task); err != nil {
        return nil, err
    }

    return task, nil
}

func (s *TaskService) Delete(userID, taskID uint) error {
    return s.repo.Delete(taskID, userID)
}
```

## 五、Handler 实现思路

三个 handler 都需要先解析路径 ID：

```go
func parseUintParam(c *gin.Context, name string) (uint, bool) {
    id, err := strconv.ParseUint(c.Param(name), 10, 64)
    if err != nil {
        response.Fail(c, http.StatusBadRequest, response.CodeInvalidRequest, "invalid id")
        return 0, false
    }
    return uint(id), true
}
```

更新接口流程：

```text
读取 user_id
解析 task id
绑定 JSON
调用 service.Update
处理 not found / invalid status / internal error
返回更新后的 task
```

删除接口流程：

```text
读取 user_id
解析 task id
调用 service.Delete
返回 deleted
```
