## 建议学习方式

每节先读一遍，再把代码复制到本地运行。遇到输出和教程不同，优先检查当前目录、Go 版本、环境变量和文件权限。日志学习最重要的是形成工程习惯：记录关键事实，避免记录敏感数据，让问题可以被复现、追踪和回滚。
