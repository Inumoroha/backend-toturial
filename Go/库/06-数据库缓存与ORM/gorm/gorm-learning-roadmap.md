## 每个阶段怎么学

每个阶段都遵循这个节奏：

1. 先读本阶段 `README.md`。
2. 再按 `01-...md`、`02-...md`、`03-...md` 的顺序学习。
3. 复制示例代码到自己的 Go 项目中运行。
4. 打开 GORM SQL 日志，观察生成的 SQL。
5. 完成本阶段练习清单。
6. 写一份学习复盘。

## 学习目标

完成整套教程后，你应该能够：

- 独立搭建 Go + GORM 后端项目。
- 根据业务设计 GORM 模型和数据库索引。
- 熟练完成 CRUD、分页、搜索、统计和聚合查询。
- 正确处理一对一、一对多、多对多关联关系。
- 使用事务保证订单、库存、标签替换等多步骤操作的一致性。
- 用 Repository、Service、Handler 分层组织项目。
- 看懂 GORM 生成的 SQL，并进行基础性能优化。
- 完成博客系统和订单系统两个项目实战。
