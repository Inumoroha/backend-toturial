## 本阶段重点

- 打印 SQL。
- 看慢查询。
- 避免全表扫描。
- 避免 N+1 查询。
- 批量插入和批量更新。
- 配置连接池。

## 本阶段最终产出

对博客系统做一次性能体检：

- 找出慢 SQL。
- 给关键字段加索引。
- 优化文章列表和详情。
- 优化批量标签创建。
