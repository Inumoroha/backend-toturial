# GORM 系统教程

这套教程面向想成为 Go 后端工程师的学习者，目标不是快速背 API，而是把 GORM 放到真实后端开发语境里系统掌握：数据库基础、模型设计、查询、关联、事务、工程分层、性能排查和项目实战。

建议学习方式：

- 按文件夹编号从 `00` 学到 `09`。
- 每个阶段先读 `README.md`，再按 `01`、`02` 的文件顺序学习。
- 每学一个 GORM API，都观察它生成的 SQL。
- 每个阶段都写代码，不要只看文档。
- 阶段练习尽量都放到同一个博客系统或订单系统里，后面会不断复用。

## 最终能力目标

完成本教程后，你应该能够：

- 独立搭建 Go + GORM 项目。
- 根据业务设计数据库表和 GORM 模型。
- 熟练完成 CRUD、条件查询、分页、聚合和复杂查询。
- 正确处理一对一、一对多、多对多关系。
- 使用事务保证多表写入的数据一致性。
- 在 Web API 项目中合理封装 GORM。
- 看懂并优化 GORM 生成的 SQL。
- 独立完成博客系统、订单系统、后台管理系统等后端项目。
