# Go strings 后端实践教程

这套教程面向想做 Go 后端开发的学习者，重点不是背 `strings` 包函数表，而是学会在真实项目里处理用户输入、URL、权限标识、日志、配置和 SQL 片段。

建议学习顺序：

1. 先读 [00-学习路线/00-strings学习路线.md](00-学习路线/00-strings学习路线.md)
2. 按编号学习 `01-输入清洗` 到 `06-项目实战`
3. 每节都把示例复制运行一遍，再按“本节达标标准”自查

本教程以 Go 1.26 的 `strings` 标准库行为为参考，示例尽量保持简单，可直接复制运行。

## 文件结构

- `00-学习路线`：学习路线、阶段目标和练习安排
- `01-输入清洗`：处理用户输入、大小写、空白字符
- `02-查找判断`：判断前缀、后缀、包含关系，解析业务字符串
- `03-切分拼接`：处理标签、权限、CSV 风格字段、展示文本
- `04-替换转换`：日志脱敏、非法字符处理、字符映射
- `05-性能与接口`：`Builder`、`Reader` 和工程边界
- `06-项目实战`：短链接参数清洗与校验小项目
