## 三、第二步：创建迁移

使用 goose：

```powershell
mkdir migrations
goose -dir migrations -s create create_short_links sql
```

迁移内容：

```sql
-- +goose Up
-- +goose StatementBegin
CREATE SCHEMA IF NOT EXISTS app;

CREATE EXTENSION IF NOT EXISTS pgcrypto;

CREATE TABLE app.short_links (
    id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
    code text NOT NULL,
    original_url text NOT NULL,
    visit_count bigint NOT NULL DEFAULT 0,
    created_at timestamptz NOT NULL DEFAULT now(),
    updated_at timestamptz NOT NULL DEFAULT now(),
    CONSTRAINT short_links_code_unique UNIQUE (code),
    CONSTRAINT short_links_code_not_empty_check CHECK (length(trim(code)) > 0),
    CONSTRAINT short_links_original_url_not_empty_check CHECK (length(trim(original_url)) > 0),
    CONSTRAINT short_links_visit_count_non_negative_check CHECK (visit_count >= 0)
);

CREATE INDEX short_links_created_at_desc_idx
ON app.short_links (created_at DESC);
-- +goose StatementEnd

-- +goose Down
-- +goose StatementBegin
DROP TABLE app.short_links;
-- +goose StatementEnd
```

执行：

```powershell
goose -dir migrations postgres $env:DATABASE_URL up
```

检查：

```sql
\d app.short_links
```

## 六、第五步：核心 SQL 清单

创建短链接：

```sql
INSERT INTO app.short_links (code, original_url)
VALUES ($1, $2)
RETURNING id, code, original_url, visit_count, created_at, updated_at;
```

跳转并增加计数：

```sql
UPDATE app.short_links
SET visit_count = visit_count + 1,
    updated_at = now()
WHERE code = $1
RETURNING original_url;
```

查询详情：

```sql
SELECT id, code, original_url, visit_count, created_at, updated_at
FROM app.short_links
WHERE code = $1;
```

分页列表：

```sql
SELECT id, code, original_url, visit_count, created_at, updated_at
FROM app.short_links
ORDER BY created_at DESC
LIMIT $1 OFFSET $2;
```

这些 SQL 应该都写在 repository 层。

## 十四、最终检查清单

完成项目后逐项检查：

- 数据库迁移文件已提交到项目。
- `app.short_links` 表存在。
- `code` 有唯一约束。
- `created_at` 有倒序索引。
- Go 服务启动成功。
- `pgxpool` 启动时能 `Ping` 成功。
- 创建短链使用参数化 SQL。
- 短码使用 `crypto/rand` 生成。
- 唯一冲突会转换为业务错误并重试。
- 跳转使用 `UPDATE ... RETURNING`。
- 访问次数并发更新不会丢失。
- 查询详情不存在时返回 `404`。
- 非法 URL 返回 `400`。
- README 能指导别人启动项目。
