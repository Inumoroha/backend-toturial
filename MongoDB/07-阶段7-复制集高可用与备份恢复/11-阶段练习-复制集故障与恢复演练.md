# 11. 阶段练习：复制集故障与恢复演练

本练习要求你完整走一遍复制集、高可用、备份恢复和变更监听。

## 1. 练习目标

完成：

- 搭建三节点复制集。
- 写入测试数据。
- 停掉 primary。
- 观察重新选主。
- 测试 majority 写入。
- 做一次 mongodump 备份。
- 删除数据后 mongorestore 恢复。
- 使用 Change Streams 监听订单变更。

## 2. 练习步骤

### 第一步：搭建复制集

按 `02-Docker-Compose搭建三节点复制集.md` 启动：

```powershell
docker compose up -d
```

初始化：

```javascript
rs.initiate({
  _id: "rs0",
  members: [
    { _id: 0, host: "mongo1:27017" },
    { _id: 1, host: "mongo2:27017" },
    { _id: 2, host: "mongo3:27017" }
  ]
})
```

### 第二步：写入数据

```javascript
use stage7_practice
db.orders.insertOne({
  order_no: "O-STAGE7-001",
  status: "pending_payment",
  total_amount: 9900,
  created_at: new Date()
})
```

### 第三步：记录 primary

```javascript
rs.status().members.map(m => ({
  name: m.name,
  stateStr: m.stateStr
}))
```

### 第四步：停掉 primary

```powershell
docker stop mongo-rs1
```

如果 primary 不是 `mongo-rs1`，换成实际容器名。

等待选举完成。

### 第五步：验证新 primary

```javascript
rs.status().members.map(m => ({
  name: m.name,
  stateStr: m.stateStr
}))
```

写入：

```javascript
db.orders.insertOne({
  order_no: "O-STAGE7-002",
  status: "paid",
  total_amount: 19900,
  created_at: new Date()
})
```

### 第六步：测试 majority

```javascript
db.orders.insertOne(
  {
    order_no: "O-STAGE7-MAJORITY",
    status: "paid",
    total_amount: 29900,
    created_at: new Date()
  },
  {
    writeConcern: { w: "majority", wtimeout: 5000 }
  }
)
```

### 第七步：备份

```powershell
mongodump `
  --uri="mongodb://localhost:27017,localhost:27018,localhost:27019/?replicaSet=rs0" `
  --db=stage7_practice `
  --out="C:\Users\HP\Desktop\mongodb-backups\stage7"
```

### 第八步：删除并恢复

```javascript
use stage7_practice
db.orders.drop()
```

恢复：

```powershell
mongorestore `
  --uri="mongodb://localhost:27017,localhost:27018,localhost:27019/?replicaSet=rs0" `
  --db=stage7_practice `
  "C:\Users\HP\Desktop\mongodb-backups\stage7\stage7_practice"
```

### 第九步：Change Streams

终端 1：

```javascript
use stage7_practice
const stream = db.orders.watch([], { fullDocument: "updateLookup" })
while (stream.hasNext()) {
  printjson(stream.next())
}
```

终端 2：

```javascript
use stage7_practice
db.orders.updateOne(
  { order_no: "O-STAGE7-001" },
  { $set: { status: "paid", updated_at: new Date() } }
)
```
