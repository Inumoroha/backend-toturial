## 3. 操作验收

你应该能独立完成下面操作，不看前面教程也能写出来。

### 创建数据

```javascript
use stage1_exam
db.dropDatabase()

db.users.insertMany([
  {
    username: "alice",
    email: "alice@example.com",
    roles: ["user", "author"],
    profile: { city: "Shanghai" },
    created_at: new Date()
  },
  {
    username: "bob",
    email: "bob@example.com",
    roles: ["user", "admin"],
    profile: { city: "Beijing" },
    created_at: new Date()
  }
])
```

### 查询数据

```javascript
db.users.find().pretty()
db.users.findOne({ username: "alice" })
db.users.find({ roles: "admin" }).pretty()
db.users.find({ "profile.city": "Shanghai" }).pretty()
db.users.find({}, { username: 1, email: 1, _id: 0 }).pretty()
```

### 更新数据

```javascript
db.users.updateOne(
  { username: "alice" },
  {
    $addToSet: { roles: "editor" },
    $set: { updated_at: new Date() }
  }
)
```

### 删除数据

```javascript
db.users.find({ username: "bob" }).pretty()
db.users.deleteOne({ username: "bob" })
```

## 4. 常用命令速查

### 数据库

```javascript
show dbs
db
use mongodb_stage1
db.dropDatabase()
```

### 集合

```javascript
show collections
db.users.drop()
```

### 插入

```javascript
db.users.insertOne({ name: "Alice" })

db.users.insertMany([
  { name: "Alice" },
  { name: "Bob" }
])
```

### 查询

```javascript
db.users.find()
db.users.find().pretty()
db.users.findOne({ name: "Alice" })
db.users.find({ age: { $gt: 20 } })
db.users.find({ status: { $in: ["active", "blocked"] } })
db.users.find({ "profile.city": "Shanghai" })
db.users.find({ roles: "admin" })
```

### 投影

```javascript
db.users.find(
  { status: "active" },
  { name: 1, email: 1, _id: 0 }
)
```

### 排序和分页

```javascript
db.posts.find().sort({ created_at: -1 })
db.posts.find().sort({ created_at: -1 }).limit(10)
db.posts.find().sort({ created_at: -1 }).skip(10).limit(10)
```

### 统计

```javascript
db.users.countDocuments()
db.users.countDocuments({ status: "active" })
```

### 更新

```javascript
db.users.updateOne(
  { email: "alice@example.com" },
  { $set: { name: "Alice New" } }
)

db.users.updateMany(
  { status: "inactive" },
  { $set: { archived: true } }
)
```

### 数组更新

```javascript
db.users.updateOne(
  { email: "alice@example.com" },
  { $push: { tags: "go" } }
)

db.users.updateOne(
  { email: "alice@example.com" },
  { $addToSet: { tags: "mongodb" } }
)

db.users.updateOne(
  { email: "alice@example.com" },
  { $pull: { tags: "go" } }
)
```

### 删除

```javascript
db.users.deleteOne({ email: "alice@example.com" })
db.users.deleteMany({ status: "inactive" })
```

### 软删除

```javascript
db.users.updateOne(
  { email: "alice@example.com" },
  { $set: { deleted_at: new Date() } }
)

db.users.find({ deleted_at: { $exists: false } })
```

### ObjectId 查询

```javascript
db.users.findOne({ _id: ObjectId("66a1f1c2e138237e2f7d0001") })
```

## 5. 常见错误速查

### 用字符串查 ObjectId

错误：

```javascript
db.users.findOne({ _id: "66a1f1c2e138237e2f7d0001" })
```

正确：

```javascript
db.users.findOne({ _id: ObjectId("66a1f1c2e138237e2f7d0001") })
```

### update 没有使用更新操作符

错误：

```javascript
db.users.updateOne(
  { email: "alice@example.com" },
  { name: "Alice New" }
)
```

正确：

```javascript
db.users.updateOne(
  { email: "alice@example.com" },
  { $set: { name: "Alice New" } }
)
```

### deleteMany 条件为空

危险：

```javascript
db.users.deleteMany({})
```

除非你明确要清空集合，否则不要这样写。

### 忘记点语法加引号

错误：

```javascript
db.users.find({ profile.city: "Shanghai" })
```

正确：

```javascript
db.users.find({ "profile.city": "Shanghai" })
```
