# 10. Go 中创建索引与性能查询

本节学习在 Go Driver v2 中创建索引、执行分页查询和使用 `BulkWrite`。

## 1. 创建单个索引

```go
func EnsureUserIndexes(ctx context.Context, collection *mongo.Collection) error {
    _, err := collection.Indexes().CreateOne(ctx, mongo.IndexModel{
        Keys: bson.D{{"email", 1}},
        Options: options.Index().
            SetUnique(true).
            SetName("uniq_users_email"),
    })
    return err
}
```

需要导入：

```go
import (
    "context"

    "go.mongodb.org/mongo-driver/v2/bson"
    "go.mongodb.org/mongo-driver/v2/mongo"
    "go.mongodb.org/mongo-driver/v2/mongo/options"
)
```

## 2. 创建多个索引

```go
func EnsureOrderIndexes(ctx context.Context, collection *mongo.Collection) error {
    models := []mongo.IndexModel{
        {
            Keys: bson.D{{"order_no", 1}},
            Options: options.Index().
                SetUnique(true).
                SetName("uniq_orders_order_no"),
        },
        {
            Keys: bson.D{{"user_id", 1}, {"created_at", -1}},
            Options: options.Index().
                SetName("idx_orders_user_created_at"),
        },
        {
            Keys: bson.D{{"user_id", 1}, {"status", 1}, {"created_at", -1}},
            Options: options.Index().
                SetName("idx_orders_user_status_created_at"),
        },
        {
            Keys: bson.D{{"status", 1}, {"created_at", -1}},
            Options: options.Index().
                SetName("idx_orders_status_created_at"),
        },
    }

    _, err := collection.Indexes().CreateMany(ctx, models)
    return err
}
```

建议：

- 服务启动或迁移阶段确保索引。
- 生产环境大集合建索引要谨慎规划。
- 复杂项目更建议使用迁移工具管理索引。

## 4. 用户订单列表查询

```go
type OrderListFilter struct {
    UserID bson.ObjectID
    Status string
    Limit  int64
}

func ListUserOrders(ctx context.Context, orders *mongo.Collection, input OrderListFilter) ([]Order, error) {
    limit := input.Limit
    if limit <= 0 || limit > 100 {
        limit = 20
    }

    filter := bson.D{{"user_id", input.UserID}}
    if input.Status != "" {
        filter = append(filter, bson.E{Key: "status", Value: input.Status})
    }

    opts := options.Find().
        SetSort(bson.D{{"created_at", -1}}).
        SetLimit(limit)

    cursor, err := orders.Find(ctx, filter, opts)
    if err != nil {
        return nil, err
    }
    defer cursor.Close(ctx)

    var results []Order
    if err := cursor.All(ctx, &results); err != nil {
        return nil, err
    }
    return results, nil
}
```

这个查询需要索引：

```javascript
{ user_id: 1, created_at: -1 }
```

或：

```javascript
{ user_id: 1, status: 1, created_at: -1 }
```

取决于是否经常带 `status`。

## 5. 游标分页查询

```go
type OrderCursor struct {
    CreatedAt time.Time
    ID        bson.ObjectID
}

type CursorOrderFilter struct {
    UserID bson.ObjectID
    Cursor *OrderCursor
    Limit  int64
}

func ListUserOrdersByCursor(ctx context.Context, orders *mongo.Collection, input CursorOrderFilter) ([]Order, error) {
    limit := input.Limit
    if limit <= 0 || limit > 100 {
        limit = 20
    }

    filter := bson.D{{"user_id", input.UserID}}

    if input.Cursor != nil {
        filter = append(filter, bson.E{
            Key: "$or",
            Value: bson.A{
                bson.D{{"created_at", bson.D{{"$lt", input.Cursor.CreatedAt}}}},
                bson.D{
                    {"created_at", input.Cursor.CreatedAt},
                    {"_id", bson.D{{"$lt", input.Cursor.ID}}},
                },
            },
        })
    }

    opts := options.Find().
        SetSort(bson.D{{"created_at", -1}, {"_id", -1}}).
        SetLimit(limit)

    cursor, err := orders.Find(ctx, filter, opts)
    if err != nil {
        return nil, err
    }
    defer cursor.Close(ctx)

    var results []Order
    if err := cursor.All(ctx, &results); err != nil {
        return nil, err
    }
    return results, nil
}
```

需要索引：

```javascript
{ user_id: 1, created_at: -1, _id: -1 }
```

## 6. BulkWrite

批量更新订单状态：

```go
func BulkMarkOrdersCancelled(ctx context.Context, orders *mongo.Collection, orderIDs []bson.ObjectID) error {
    models := make([]mongo.WriteModel, 0, len(orderIDs))

    now := time.Now().UTC()
    for _, id := range orderIDs {
        models = append(models, mongo.NewUpdateOneModel().
            SetFilter(bson.D{{"_id", id}, {"status", "pending_payment"}}).
            SetUpdate(bson.D{{"$set", bson.D{
                {"status", "cancelled"},
                {"updated_at", now},
            }}}),
        )
    }

    if len(models) == 0 {
        return nil
    }

    opts := options.BulkWrite().SetOrdered(false)
    _, err := orders.BulkWrite(ctx, models, opts)
    return err
}
```

需要导入：

```go
import "time"
```

## 7. Go 中不要忽略 explain

Go Driver 也可以执行 explain 命令，但学习阶段建议先用 `mongosh` 分析查询，再把索引和查询逻辑写入 Go。

核心思想：

```text
Go 代码负责稳定生成查询。
mongosh/explain 负责验证查询计划。
```

## 9. 本节小结

你需要记住：

- Go 中通过 `collection.Indexes().CreateOne/CreateMany` 创建索引。
- 索引名建议显式设置。
- 游标分页需要稳定排序和匹配索引。
- `BulkWrite` 使用 `mongo.WriteModel`。
- 生产环境复杂索引建议迁移工具管理。
