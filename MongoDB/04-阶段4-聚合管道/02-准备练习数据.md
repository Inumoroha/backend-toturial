# 02. 准备练习数据

本阶段所有示例都基于同一套数据。建议你先完整执行本文件中的初始化脚本，后面每一节都可以直接运行。

## 1. 初始化数据库

连接 MongoDB：

```powershell
mongosh "mongodb://root:example@localhost:27017/?authSource=admin"
```

执行：

```javascript
use aggregation_stage4
db.dropDatabase()
use aggregation_stage4
```

## 8. 建议索引

这些索引先建上，后面第 5 阶段会解释为什么这样设计。

```javascript
db.orders.createIndex({ status: 1, paid_at: -1 })
db.orders.createIndex({ user_id: 1, created_at: -1 })
db.orders.createIndex({ "items.product_id": 1 })
db.comments.createIndex({ post_id: 1, created_at: -1 })
db.events.createIndex({ user_id: 1, created_at: -1 })
```
